# check Command

!!! warning "Deprecated"
    `csttool check` is deprecated. Use [`csttool doctor`](doctor.md) instead — it performs all the same checks plus dcm2niix version validation, atlas data verification, writable directory checks, and PDF backend diagnostics.

    `csttool check` redirects to `csttool doctor` automatically and will be removed in a future release.

## Usage

```bash
csttool check   # redirects to csttool doctor
```
